Configuring the Heroku production environment for Laravel

  1. In the Heroku dashboard:
    • In the config vars, set APP_ENV = production
  2. In Laravel:
    • In app/Providers/AppServiceProvider.php, add the following code to the boot() method to enforce connection via HTTPS at all times: 
      if(env('APP_ENV') == 'production') {
    \URL::forceScheme('https');
}
    • Use the asset() Laravel helper function to generate urls to CSS and JS files in your HTML.
    • Before pushing your code to Heroku, run npm run prod to have Laravel package your code for a production environment.
