Terminology

Unary Operators
Operators that act on one (and only one) value
Ex: i++, i--
Binary Operators
Operators that act on two values
Ex: x + y, x % y
THE Ternary Operator
An operator that acts on three values. There's only one:
x ? y : z
This is the same as 
if (x) {
    y;
} else {
    z;
}
(See this explanation—thanks, Phillip!)
