  • CSS shorthand format, for example:
    • margin: 10px;
      (sets 10px margin on all sides of an element)
    • margin: 10px 20px;
      (sets 10px margin on top and bottom, 20px margin on left and right sides)
    • margin: 10px 20px 30px;
      (sets 10px top margin, 20px left and right margins, and 30px bottom margin)
    • margin: 10px, 20px, 30px, 40px;
      (sets 10px top, 20px right, 30px bottom, and 40px left margins)
  • JavaScript topics
    • this
    • two ways to set event handlers:
      1. item.onclick = function;
      2. item.addEventHandler("click", function);

      • and DOES allow bracket notation.
    • DOES NOT WORK: A == B == C. Instead, use A == B && A == C
